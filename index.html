Games - Rock, Paper Scissor

In this project I built a rock-paper-scissor game using HTML, CSS, and Javascript. This game is a testament to the various skills I have learned in web development, from HTML and CSS styling to implementing game logic using Javascript and DOM manipulation.

One of the key challenges in building this game was designing a user-friendly interface that would allow players to easily select their choice of rock, paper, or scissors. To achieve this, I utilized CSS to create a sleek and modern design that highlighted the player's selected choice and displayed the results of each game round.

Another challenge was implementing the game logic using Javascript. I utilized a range of techniques, from writing functions to generate random computer choices to using if/else statements to determine the winner of each game round. I also incorporated DOM manipulation to dynamically update the page with the results of each game and keep track of the score.

Throughout the development process, I drew upon my knowledge of web development, including HTML, CSS, and Javascript, to create a dynamic and engaging game. I utilized responsive design techniques, such as media queries, to ensure that the game was accessible and user-friendly on a range of devices and screen sizes.

This game is a testament to the many skills I have learned, including HTML, CSS, and Javascript, as well as the importance of designing user-friendly interfaces and implementing game logic using DOM manipulation.
